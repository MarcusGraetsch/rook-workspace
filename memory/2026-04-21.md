# Session 2026-04-21: IDP Plattform Build

## Was wir gebaut haben

Eine komplette Internal Developer Platform (IDP) auf Kubernetes (kind) mit:
- Flux GitOps
- ArgoCD UI
- Keycloak OIDC Provider
- OPA Gatekeeper (Policy Enforcement)
- Trivy (CVE Scanning)
- kube-bench (CIS/BSI Compliance)
- Polaris (Best Practice Validation)
- Prometheus + Grafana
- midPoint (Identity Governance)
- SOPS + Age (Secrets Management)
- GitHub Actions CI/CD Pipeline
