## 🚀 GitHub: Release 2026.4.22 (23. April)

Großes Release mit Fokus auf **Voice/Audio**, **Provider-Erweiterungen** und **UX-Verbesserungen**.

**Highlights:**

- **xAI-Provider ausgebaut:** Bildgenerierung (grok-imagine-image / -pro), Text-to-Speech mit 6 Live-Stimmen, Speech-to-Text, Realtime-Transkription für Voice Calls — alles via xAI.
- **Voice Call Streaming STT:** Deepgram, ElevenLabs (inkl. Scribe v2) und Mistral bekommen Echtzeit-Sprachtranskription, dazu OpenAI und xAI bestehend.
- **TUI Embedded Mode:** Terminal-Chats laufen jetzt lokal ohne Gateway, behalten aber Plugin-Approval-Mechanismen bei.
- **Onboarding smarter:** Fehlende Provider/Channel-Plugins werden beim Setup automatisch installiert.
- **OpenAI Responses:** Nutzt native `web_search` von OpenAI, wenn keine explizite Such-Provider gepinnt ist (Brave etc. bleiben managed).
- **`/models add`:** Neue Modelle können direkt aus dem Chat registriert werden — kein Gateway-Restart nötig.
- **WhatsApp verbessert:** Native Reply-Quoting (`replyToMode`) + per-Gruppe/per-Direct `systemPrompt` Konfiguration mit Wildcard-Support.
- **Sessions/Mailbox:** `sessions_list` jetzt filterbar nach Label, Agent und Suche — mit Vorschau von Titel und letzter Nachricht.
- **Control UI Personalisierung:** Browser-lokale Operator-Identität (Name + Avatar), bessere Mobile-Layouts.
- **Gateway Diagnostics:** Stabilitäts-Recording standardmäßig aktiv + Export-Funktion für Support-Bugreports (sanitized Logs, Config, Health).
- **Neue Provider:** Tencent Cloud (TokenHub, hy3-preview Modelle) + Amazon Bedrock Mantle (Claude Opus 4.7 via bearer-auth Streaming).
- **GPT-5 Overlay:** Prompt-Verhalten und Heartbeat-Guidance für GPT-5-Modelle jetzt im Shared Provider Runtime — funktioniert über OpenAI, OpenRouter, OpenCode, Codex etc.
