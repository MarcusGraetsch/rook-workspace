## 1. Was ist "Fractional DevOps"?

**Definition:** Ein erfahrener DevOps-Engineer arbeitet 1–3 Tage/Woche für ein Startup, das keine Vollzeit-DevOps-Stelle braucht (oder sich keine leisten kann).

**Dein Wert:**
- Aufbau der kompletten Infrastruktur (von 0 zu Production)
- CI/CD-Pipelines, Monitoring, Security
- Team-Coaching und Best Practices
- Skalierungsstrategie für Wachstum

**Warum Startups das brauchen:**
- Gründer können Code, aber keine Infrastruktur
- AWS-Rechnungen explodieren ohne Optimierung
- Security wird erst nach dem Hack wichtig
- Developer-Productivity leidet unter manuellen Deployments

## 2. Dein Angebotspaket

### Paket A: "Foundation" (2 Tage/Woche, 3 Monate)
**Preis:** €6.000–€8.000/Monat

**Inhalt:**
- K8s-Cluster-Setup (Cloud oder On-Prem)
- CI/CD-Pipeline (GitLab CI oder GitHub Actions)
- Monitoring-Stack (Prometheus, Grafana, Alertmanager)
- Backup-Strategie
- Dokumentation für das Team
- Security-Basics (Secrets Management, RBAC)

**Ziel:** Produktionsreife Infrastruktur, die das Team selbst betreiben kann.

---

### Paket B: "Scale" (1–2 Tage/Woche, ongoing)
**Preis:** €4.000–€6.000/Monat

**Inhalt:**
- Alles aus Paket A
- Performance-Optimierung
- Cost-Optimization (Cloud-Sparen)
- Incident Response
- On-Call-Bereitschaft (optional, mit Zuschlag)
- Neue Features: Multi-Cluster, GitOps, Service Mesh

**Ziel:** Infrastruktur wächst mit dem Unternehmen.

---

### Paket C: "Rescue" (Projektbasiert, 2–4 Wochen)
**Preis:** €10.000–€20.000 (einmalig)

**Inhalt:**
- Audit der bestehenden Infrastruktur
- Notfall-Fixes (Performance, Security, Downtime)
- Refactoring-Plan
- Knowledge-Transfer an das Team

**Ziel:** Chaos beheben, stabile Basis schaffen.

### Must-Have Klauseln

| Klausel | Beschreibung |
|---------|-------------|
| **Scope** | Konkrete Deliverables, keine endlosen "kleinen Anfragen" |
| **Zeitbudget** | Feste Tage/Woche, keine Überstunden ohne Absprache |
| **Kündigungsfrist** | 1 Monat für beide Seiten |
| **On-Call** | Optional, extra vergütet, definierte Reaktionszeiten |
| **IP & Haftung** | Keine Haftung für Downtime, keine IP-Übertragung |
| **Exklusivität** | Keine Konkurrenz im gleichen Marktsegment |
| **HiSolutions** | Keine Kundenabwerbung, keine Konkurrenzprodukte |

### Preisgestaltung

- **Tagessatz:** €800–€1.200 (je nach Komplexität)
